# 📚 소설 개인 보관함 프로그램 (Novel Bookshelf)

## 📖 프로젝트 소개

Java Swing을 이용하여 제작한 '개인용 소설 텍스트 파일 통합 관리 시스템'입니다.

사용자는 로컬 하드디스크에 보유 중인 소설 텍스트 회차 파일들을 프로그램에 연동하고, 로컬 텍스트 파일을 편리하게 열람할 수 있습니다.

---

# ⚠️필독⚠️

### 💡본 프로그램은 대용량 소설 데이터 축적 시의 파일 관리 편의성과 확장성을 극대화하기 위해, 프로젝트 내부가 아닌 로컬 디스크의 **독립된 외부 절대 경로(`C:\novel`)**에서 리소스를 실시간 참조합니다. 처음 실행  필요한 기본 구조를 아래와 같이 세팅해 주시기 바랍니다.


### 1. 필수 초기 세팅
#### **프로그램이 정상적으로 작동하려면 C:\novel 경로에 아래의 필수 파일들이 배치되어야 합니다.**

- icon 폴더 (필수): 시스템 UI 및 로고 표시를 위해 필요합니다.
해당 폴더 내에 decor.png, decor2.png, no_cover.png, kakao.png, joara.png, munpia.png, series.png 파일이 위치해야 합니다.

참고: icon 폴더가 없으면 일부 아이콘이 정상적으로 출력되지 않을 수 있습니다.

#### 📁 1. 로컬 환경 독립 폴더 구조 세팅 (C:\드라이브 기준)
컴퓨터의 `C:\` 최상위 경로에 `novel` 폴더를 생성하고, 그 내부에 아래와 같이 3개의 서브 폴더를 분리하여 구성합니다.

```text
C:\
└─ novel\
   ├─ icon\                        # [필수] 시스템 고유 로고 및 기본 일러스트 저장소
   │  ├─ decor.png                 # 보관함 하단 데코레이션 일러스트
   │  ├─ decor2.png                # 등록창 하단 데코레이션 일러스트
   │  ├─ no_cover.png              # 등록된 표지가 없을 때 출력되는 기본 대체 이미지
   │  ├─ kakao.png                 # 카카오페이지 로고 이미지
   │  ├─ joara.png                 # 조아라 로고 이미지
   │  ├─ munpia.png                # 문피아 로고 이미지
   │  └─ series.png                # 네이버 시리즈 로고 이미지
   │
   ├─ covers\                      # [선택] 본편 소설 커스텀 표지 이미지 모음
   │  ├─ 소설A_표지.jpg
   │  └─ 소설B_표지.png
   │
   ├─ novels\                      # [신규] 본편/패러디/단편 텍스트 본문을 유형별로 분리 보관
   │  ├─ novel_list\               # [선택] 본편 소설 본문 텍스트(.txt) 회차 폴더 보관소
   │  │  ├─ 소설A_폴더\
   │  │  └─ 소설B_폴더\
   │  │
   │  ├─ parodies\                 # [신규/선택] 패러디 작품 본문 텍스트 폴더 보관소
   │  │  ├─ 패러디A_폴더\
   │  │  └─ 패러디B_폴더\
   │  │
   │  └─ short_stories\            # [신규/선택] 단편/썰 본문 텍스트 폴더 보관소 (원작명 > 유형별 하위 분류)
   │     └─ 원작명\
   │        ├─ 단편\
   │        ├─ 썰\
   │        └─ 연작\
   │
   ├─ library_data.txt             # (자동 생성) 본편 서재 데이터 파일
   ├─ parody_shorts_data.txt       # (자동 생성) 패러디 서재 데이터 파일
   ├─ shorts_data.txt              # (자동 생성) 단편/썰 서재 데이터 파일
   ├─ work_links.txt               # (자동 생성) 단편/썰 원작 연결 정보
   ├─ platforms.txt                # (자동 생성) 단편/썰 전용 커스텀 플랫폼 목록
   ├─ favorite_shorts.txt          # (자동 생성) 단편/썰 즐겨찾기 정보
   └─ app_settings.txt             # (자동 생성) 환경설정 파일

```
### ⚠️icon폴더를 novel폴더 내부에 위치시킵니다.

## ✨ 주요 기능

-  **소설 등록 및 관리**
  제목, 작가, 장르, 키워드, 표지 이미지를 등록하고 작품 정보를 자유롭게 수정할 수 있습니다.

-  **플랫폼 분류**
  기본 등록된 유/무료 웹소설 플랫폼별로 작품을 분류하고, 사용자가 원하는 플랫폼을 무제한으로 추가할 수 있습니다.

-  **검색 및 정렬**
  제목, 작가, 키워드 기반 검색과 최근 읽은 순, 제목순, 안 읽은 작품 순 등 다양한 정렬 기능을 제공합니다.

-  **좋아요 기능**
  즐겨찾는 작품을 좋아요 목록으로 지정하여 별도로 편리하게 관리할 수 있습니다.

-  **텍스트 뷰어**
  로컬 하드디스크의 텍스트 파일을 회차별로 깨끗하게 불러와 편리하게 열람할 수 있습니다.

-  **이어보기 및 책갈피**
  마지막으로 읽은 회차를 자동으로 저장하고, 나만의 독서 메모와 함께 책갈피를 스마트하게 관리할 수 있습니다.

-  **뷰어 환경 설정**
  글꼴, 글자 크기, 줄간격, 배경 테마를 사용자의 시력과 취향에 맞게 실시간으로 변경할 수 있습니다.

-  **표지 확대**
  등록한 소설 표지를 마우스로 클릭하면 원본 고해상도 크기로 크게 펼쳐서 확인할 수 있습니다.

-  **설정 자동 저장**
  사용자의 환경 설정과 작품 메타데이터 정보를 로컬 파일에 안전하게 실시간 자동 저장합니다.

## 📋 사용 방법

### 1.  작품 등록
1. 메인 서재 화면 상단의 **`+ 새 소설 추가`** 버튼을 클릭합니다.
2. 팝업창에서 소설의 제목, 작가, 장르, #해시태그 키워드 등의 정보를 입력합니다.
3. 소설 본문(.txt) 파일들이 저장된 하드디스크의 **텍스트 폴더 경로**를 선택합니다.
4. 유저의 선택에 따라 소설 표지 이미지를 등록합니다. *(등록하지 않으면 기본 표지로 대체됩니다.)*
5. 하단의 **`저장`** 버튼을 누르면 내 서재 보관함에 작품 카드가 최종 추가됩니다.

### 2.  작품 탐색
1. 상단 검색창에 찾고자 하는 소설의 제목, 작가 이름, 또는 키워드를 입력하여 실시간 검색합니다.
2. 좌측 상단의 **`전체 플랫폼`** 필터를 선택하여 특정 플랫폼 출신 소설들만 필터링합니다.
3. 정렬 콤보박스를 통해 최근 읽은 순, 제목순, 안 읽은 작품 순 등으로 목록을 자유롭게 정렬합니다.

### 3.  작품 읽기
1. 서재 보관함에서 읽고자 하는 소설 카드를 마우스로 선택합니다.
2. 작품 상세 정보창이 열리면 컴팩트하게 정렬된 회차 목록을 확인합니다.
3. 하단의 **`▶ 이어보기`** 버튼을 누르거나, 목록에서 읽고 싶은 특정 회차를 선택합니다.
4. 텍스트 뷰어 내부의 **`이전화 / 다음화`** 버튼을 이용해 회차를 편리하게 이동하며 독서합니다.
5. 독서 중 기억하고 싶은 구역이 있다면 **책갈피 및 회차별 요약 메모 기능**을 가동합니다.
6. 우측 상단의 **`설정`** 버튼을 눌러 글꼴, 글자 크기, 줄간격, 배경 테마를 눈이 편한 상태로 조절합니다.

###  4. 환경 설정
1. 보관함 메인 화면 상단의 **`환경 설정`** 아이콘을 선택하여 진입합니다.
2. 프로그램을 처음 시작할 때 적용될 기본 화면 테마, 폰트 종류, 기본 글자 크기를 사전 지정합니다.
3. 나만의 가변 확장 플랫폼 메뉴 구축을 위해 새로운 플랫폼 이름을 등록하거나 플랫폼 아이콘 이미지를 추가/삭제합니다.
4. 변경 완료 후 **`저장`** 버튼을 누르면 설정 데이터가 로컬 스토리지에 즉시 동기화 백업됩니다.

#### 2차 제작 기획
<details>
<summary>🔍 여기를 클릭하면 2차 기획 목표 목록이 펼쳐집니다.</summary>


2차 수정

1. 보관함 창에서 출신 플랫폼을 글자가 아닌 해당 플랫폼 대표이미지로 하기❌
2. 소설 정보 창의 목록에서 '소설 제목 - 텍스트 파일 이름'으로 되어 있는 것을 '텍스트 파일 이름'만 나오게 바꾸기✔️
3. 보관함의 환경 설정 기능 활성화❌
4. 이어보기 색(현재 연두색)을 다른 색으로 바꾸기✔️
5. 소설 정보창에서 표지 클릭하면 표지 크고 선명하게 보이게 하기❌
6. 보관함 [내 서재]일 때, 상단 좌측에 전체 작품 수 표시 하기(최근 읽은 순, 제목순 = 전체 작품, 안 읽은 작품 = 안 읽은 작품 개수)✔️
7. 보관함 [좋아요]일 때, 상단 좌측에 좋아요 누른 작품 수 표시하기✔️
8. 키워드 검색 가능하게 만들기(여러 개 검색할 수 있게 해야함. ex. #판타지, #약피폐, #힐링)✔️
9. 완독 작품 특징 표시하기(색을 진하게 만든다거나 특정 표시를 준다던가)❌
