# 📚 소설 개인 보관함 프로그램 (Novel Bookshelf)

## 📖 프로젝트 소개

Java Swing을 이용하여 제작한 '개인용 소설 텍스트 파일 통합 관리 시스템'입니다.

사용자는 로컬 하드디스크에 보유 중인 소설 텍스트 회차 파일들을 프로그램에 연동하고, 로컬 텍스트 파일을 편리하게 열람할 수 있습니다.

---

## ✨ 주요 기능

- 📝 **소설 등록 및 관리**
  제목, 작가, 장르, 키워드, 표지 이미지를 등록하고 작품 정보를 자유롭게 수정할 수 있습니다.

- 🏷️ **플랫폼 분류**
  기본 등록된 유/무료 웹소설 플랫폼별로 작품을 분류하고, 사용자가 원하는 플랫폼을 무제한으로 추가할 수 있습니다.

- 🔍 **검색 및 정렬**
  제목, 작가, 키워드 기반 검색과 최근 읽은 순, 제목순, 안 읽은 작품 순 등 다양한 정렬 기능을 제공합니다.

- ❤️ **좋아요 기능**
  즐겨찾는 작품을 좋아요 목록으로 지정하여 별도로 편리하게 관리할 수 있습니다.

- 📖 **텍스트 뷰어**
  로컬 하드디스크의 텍스트 파일을 회차별로 깨끗하게 불러와 편리하게 열람할 수 있습니다.

- 📌 **이어보기 및 책갈피**
  마지막으로 읽은 회차를 자동으로 저장하고, 나만의 독서 메모와 함께 책갈피를 스마트하게 관리할 수 있습니다.

- ⚙️ **뷰어 환경 설정**
  글꼴, 글자 크기, 줄간격, 배경 테마를 사용자의 시력과 취향에 맞게 실시간으로 변경할 수 있습니다.

- 🖼️ **표지 확대**
  등록한 소설 표지를 마우스로 클릭하면 원본 고해상도 크기로 크게 펼쳐서 확인할 수 있습니다.

- 💾 **설정 자동 저장**
  사용자의 환경 설정과 작품 메타데이터 정보를 로컬 파일에 안전하게 실시간 자동 저장합니다.

---

## 📋 사용 방법

### 1. 📝 작품 등록
1. 메인 서재 화면 상단의 **`+ 새 소설 추가`** 버튼을 클릭합니다.
2. 팝업창에서 소설의 제목, 작가, 장르, #해시태그 키워드 등의 정보를 입력합니다.
3. 소설 본문(.txt) 파일들이 저장된 하드디스크의 **텍스트 폴더 경로**를 선택합니다.
4. 유저의 선택에 따라 소설 표지 이미지를 등록합니다. *(등록하지 않으면 기본 표지로 대체됩니다.)*
5. 하단의 **`저장`** 버튼을 누르면 내 서재 보관함에 작품 카드가 최종 추가됩니다.

### 2. 🔍 작품 탐색
1. 상단 검색창에 찾고자 하는 소설의 제목, 작가 이름, 또는 키워드를 입력하여 실시간 검색합니다.
2. 좌측 상단의 **`전체 플랫폼`** 필터를 선택하여 특정 플랫폼 출신 소설들만 필터링합니다.
3. 정렬 콤보박스를 통해 최근 읽은 순, 제목순, 안 읽은 작품 순 등으로 목록을 자유롭게 정렬합니다.

### 3. 📖 작품 읽기
1. 서재 보관함에서 읽고자 하는 소설 카드를 마우스로 선택합니다.
2. 작품 상세 정보창이 열리면 컴팩트하게 정렬된 회차 목록을 확인합니다.
3. 하단의 **`▶ 이어보기`** 버튼을 누르거나, 목록에서 읽고 싶은 특정 회차를 선택합니다.
4. 텍스트 뷰어 내부의 **`이전화 / 다음화`** 버튼을 이용해 회차를 편리하게 이동하며 독서합니다.
5. 독서 중 기억하고 싶은 구역이 있다면 **책갈피 및 회차별 요약 메모 기능**을 가동합니다.
6. 우측 상단의 **`설정`** 버튼을 눌러 글꼴, 글자 크기, 줄간격, 배경 테마를 눈이 편한 상태로 조절합니다.

### ⚙️ 4. 환경 설정
1. 보관함 메인 화면 상단의 **`환경 설정`** 아이콘을 선택하여 진입합니다.
2. 프로그램을 처음 시작할 때 적용될 기본 화면 테마, 폰트 종류, 기본 글자 크기를 사전 지정합니다.
3. 나만의 가변 확장 플랫폼 메뉴 구축을 위해 새로운 플랫폼 이름을 등록하거나 플랫폼 아이콘 이미지를 추가/삭제합니다.
4. 변경 완료 후 **`저장`** 버튼을 누르면 설정 데이터가 로컬 스토리지에 즉시 동기화 백업됩니다.

## 📂 프로젝트 구조
```text
src
├── Main.java
├── BookShelfPage.java
├── NovelDetailPage.java
├── AddNovelDialog.java
├── AppSettings.java
└── Novel.java
```


### Main.java
프로그램 가동 시 메인 보관함 객체를 생성하고 주 프레임 화면을 출력하는 시작점 클래스

### BookShelfPage.java
전체 작품 카드 배치, 하트(즐겨찾기) 토글 소팅 및 키워드 태그 기반 실시간 검색 처리

### AddNovelDialog.java
소설 메타데이터 입력 다이얼로그 및 7개 이상 플랫폼 탐지 시 독립 팝업 레이어 구동

### Novel.java
제목, 작가, 장르, 플랫폼, 폴더 경로, 표지 주소, 키워드, 소개글 등 개별 소설 개체의 속성 데이터 정의

### NovelDetailPage.java
소설 텍스트 폴더 정밀 스캔, 책갈피 메모 리스킨 뷰 제공 및 정렬 필터 연동 제어

### NovelViewerPage.java
소설 본문 텍스트 뷰어 화면 및 독서 편의 기능 제어

### AppSettings.java
유저가 직접 등록하고 사용하는 플랫폼 리스트 보존 및 설정 데이터 연동

### 1. 메인 서재 보관함 화면


사용자가 등록한 소설 카드들이 플랫하게 나열되며 완결 배지 및 플랫폼 로고가 정밀 도킹 출력됩니다.
